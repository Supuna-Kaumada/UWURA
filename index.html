UWU Resource Allocation System

Make Your Reservations Easier & Faster

This Website is targeted on automating the current process of allocating resources in Uva Wellassa University instead of current manual, paper-based resource allocating process. In the university, resources include lecture halls, sounds and multimedia equipment, auditorium, vehicles, and lab equipment. All the above resources can be allocated from this web based system. When it comes to the efficiency, it is very efficient process to complete a reservation.
